# 🎧 Headset Inventory Management System

BPO Headset Allocation and Inventory Management System for ABSS Pvt. Ltd.

## 📋 Features

- **Inventory Management**: Track all headsets (Voix, Tech, OJT, Y-Jack)
- **Agent Assignment**: Assign headsets to agents with deposit collection
- **Deposit Management**: Rs 1750 (Voix), Rs 1250 (Tech), Rs 500 (Process Change)
- **Repair Tracking**: Send/receive headsets for repair
- **Transfer Management**: Transfer headsets between agents
- **Process Change**: Handle Tech → Premium process upgrades
- **Digital Signatures**: Capture signatures on touchscreen
- **PDF Generation**: Generate signed deposit/assignment forms
- **Reports**: Monthly reports, Excel/PDF exports
- **Role-Based Access**: Admin, IT Staff, Manager, TL, Trainer, Agent
